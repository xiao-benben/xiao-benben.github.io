3 PATH 环境变量

我们在任意位置运行程序，但是代运行的程序往往不是在当前目录，这时为了方便，提出PATH。

PATH 用户保存可以搜索的目录路径，我们只需要将路径加入到PATH变量中，就可以在任意位置运行。

echo $PATH查看shell将到哪些目录下寻找命令或程序，PATH的值是一系列的目录

4 which

which指令会在环境变量$PATH设置的目录里搜索某个系统命令的位置，用于查看某个系统命令是否存在，以及执行的哪个位置的命令。
